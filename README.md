# docker
## Install

Refer to https://docs.docker.com/engine/install/ubuntu/

# Script

This script is optimized to ubuntu 20.04

## Install minikube

````
./script.sh install
````
## Start

````
./script.sh start
````

## Install Fluxctl

```
./script.sh flux
```
## Remove

```
./script.sh remove
```
## Flux identity

```
fluxctl identity
```
